MPMS v3 Uni / team

Built for a university research client studying clinician attention and multitasking in high-pressure environments. The app simulates dozens of concurrent patients simultaneously, each with live waveform animations running at 60fps and microsecond-accuracy interaction logging to capture exactly when and how clinicians respond to alerts.

Implemented a binaural audio alert system using the Web Audio API — different sounds in each ear to help clinicians distinguish between patients without looking. Per-patient configurable thresholds, drag-and-drop layout editor, and a medical-grade UI.

Led a 6-person Agile team.

Raspberry Pi plant waterer In progress

A real-time soil moisture monitoring and automated irrigation system running on a Raspberry Pi Zero 2W. Uses two STEMMA soil sensors feeding into a 4-channel relay module controlling four mini submersible pumps. Designed to water plants only when soil moisture drops below a set threshold, targeting meaningful water reduction over a timer-based approach.

Hardware assembly underway — replacement Pi pending after a dead-on-arrival unit.
